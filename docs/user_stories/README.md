# User stories

1. [As a user, I want to customize a kart so that I can see its final features.](./user_story_01.md)

2. [As a user, I want to be able to change the UI language so that I can understand clearly the UI.](./user_story_02.md)

3. [As a user, I want to get a list of customized karts when I choose some kart features so that I can know which combinations fulfills that features.](./user_story_03.md)

4. [As a user, when I choose a circuit I want to get a list of customized karts so that I can find a suitable kart to compete in that circuit.](./user_story_04.md)

5. [As a user, when I choose a Grand Prix I want to get a list of customized karts so that I can find a suitable kart to compete in that Grand Prix.](./user_story_05.md)
